# supermarket-sales-analysis
Analisis penjualan supermarket berdasarkan transaksi nyata dari 3 cabang di Myanmar. Project ini mencakup EDA, visualisasi Excel, dan insight bisnis.

## 1. Overview
Dataset ini berisi 1.000 transaksi dari sebuah supermarket chain dengan 3 cabang.
Tujuan project ini adalah:
- Memahami pola penjualan
- Mengidentifikasi kategori produk terlaris
- Melihat jam dan hari paling ramai
- Mengetahui cabang paling menguntungkan
- Memberikan insight bisnis untuk strategi pemasaran

## 2. Dataset
Sumber dataset: Kaggle (https://www.kaggle.com/datasets/faresashraf1001/supermarket-sales)
Rows: 1000
Columns: 17
Berisi informasi transaksi lengkap seperti:
- Branch, City
- Customer Type, Gender
- Product Line
- Unit Price, Quantity
- Total, COGS, Gross Income
- Date, Time, Payment
- Rating

## 3. Tools & Technology
- Excel → Cleaning, Pivot Table, Dashboard

## 4. Workflow
- Import & cleaning data
- Membuat pivot table

## 6. Business Summary
- Jam memuncak: 12–15
- Hari terbaik: Sabtu
- Kategori terlaris: Food & Beverages
- Cabang paling profitable: C
- Pembelian meningkat saat siang maka cocok untuk promo lunch time

## 7. Final Conclusion

Dataset menunjukkan bahwa food & beverages adalah andalan supermarket, aktivitas belanja lebih ramai di siang hari dan akhir pekan, dengan branch C sebagai lokasi terbaik.
Strategi marketing & manajemen stok sebaiknya difokuskan ke kategori dan waktu puncak tersebut.
